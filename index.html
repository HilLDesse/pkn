Minggu 1
Pengantar SPL & Matriks

Identifikasi persamaan linear, matriks augmentasi, dan operasi matriks dasar

📖 Materi: Persamaan Linear

Definisi

Persamaan linear adalah persamaan di mana semua variabelnya berpangkat 1 dan tidak ada operasi seperti perkalian antar variabel, sin, cos, akar, atau eksponen variabel.

Bentuk Umum Persamaan Linear

a₁x₁ + a₂x₂ + ... + aₙxₙ = b

di mana: aᵢ = koefisien (konstanta), xᵢ = variabel, b = konstanta

Yang BUKAN Persamaan Linear

  • Pangkat variabel ≠ 1 → contoh: x², x³, √x
  • Perkalian antar variabel → contoh: x₁·x₂
  • Fungsi trigonometri/logaritma → contoh: sin(x), ln(x), eˣ
📖 Materi: Matriks Augmentasi

Cara Membentuk

Matriks augmentasi adalah cara menuliskan SPL dalam bentuk matriks. Koefisien kiri disusun sebagai matriks, lalu dipisah dengan garis vertikal, diikuti kolom konstanta kanan.

Contoh Konversi

2x + 3y = 7 [ 2 3 | 7 ]
x - y = 1 → [ 1 -1 | 1 ]
📖 Materi: Operasi Matriks Dasar

Penjumlahan & Pengurangan Matriks

A + B = [aᵢⱼ + bᵢⱼ] (elemen demi elemen, ordo sama)
kA = [k·aᵢⱼ] (skalar dikali setiap elemen)
✏️ Soal Kisi-Kisi + Pembahasan Lengkap

Manakah yang tergolong persamaan linear? Berikan alasan.

    2x₁ − 3x₂ + 4x₃ = 12 → ✅ PERSAMAAN LINEAR

    Variabel x₁ memiliki pangkat 2 (x₁²). Persamaan linear mensyaratkan semua variabel berpangkat tepat 1.

    Terdapat fungsi sinus sin(x₂). Fungsi trigonometri bersifat nonlinear — tidak dapat ditulis dalam bentuk koefisien × variabel.

Minggu 2
Eliminasi Gauss & Gauss-Jordan

Operasi Baris Elementer (OBE), Bentuk Eselon Baris, pemecahan SPL

📖 Materi: Operasi Baris Elementer (OBE)

3 Jenis OBE yang Diperbolehkan

  1. Tukar baris: Rᵢ ↔ Rⱼ  (menukar posisi dua baris)
  2. Kalikan baris dengan konstanta ≠ 0: kRᵢ → Rᵢ
  3. Tambah kelipatan baris lain: Rᵢ + kRⱼ → Rᵢ  (paling sering dipakai!)
📖 Materi: Bentuk Eselon Baris (REF)

Syarat REF (Row Echelon Form)

  1. Semua baris nol (jika ada) berada di paling bawah
  2. Elemen pertama tak nol tiap baris disebut pivot (leading 1)
  3. Pivot setiap baris berada di sebelah kanan pivot baris di atasnya
  4. Semua elemen di bawah pivot = 0

Contoh REF vs Non-REF

REF ✅: Non-REF ❌ (pivot tidak naik ke kanan):
[1 2 3] [1 0 2]
[0 1 4] [0 0 1] ← pivot di kol 3
[0 0 1] [0 1 0] ← pivot di kol 2, tapi LEBIH RENDAH
📖 Materi: Substitusi Balik

Langkah Substitusi Balik

  1. Mulai dari baris paling bawah → langsung dapatkan nilai variabel terakhir
  2. Substitusikan ke baris di atasnya untuk mendapat variabel berikutnya
  3. Ulangi ke atas hingga semua variabel ditemukan
Easy Soal 1 — Identifikasi REF
Matriks mana yang sudah REF?
A = [[1,2,3],[0,1,4],[0,0,1]]  |  B = [[1,0,2],[0,0,1],[0,1,0]]
  • A
    Matriks A → ✅ Sudah REF
    Pivot baris 1: posisi (1,1) = 1
    Pivot baris 2: posisi (2,2) = 1 → lebih kanan dari baris 1 ✓
    Pivot baris 3: posisi (3,3) = 1 → lebih kanan dari baris 2 ✓
    Semua elemen di bawah pivot = 0 ✓
  • B
    Matriks B → ❌ Bukan REF
    Pivot baris 2: posisi (2,3) = 1 (kolom 3)
    Pivot baris 3: posisi (3,2) = 1 (kolom 2)
    Masalah: Pivot baris 3 (kolom 2) lebih KIRI dari pivot baris 2 (kolom 3). Ini melanggar syarat REF di mana pivot harus terus bergeser ke kanan.

Operasi Matriks Lanjutan & Invers

Perkalian matriks berbeda ordo, transpose, dan invers 2×2

📖 Materi: Perkalian Matriks

Syarat & Ordo Hasil

A (m×n) × B (n×p) = C (m×p)

Syarat: Kolom A harus = Baris B

Cara Hitung Elemen Cᵢⱼ

Cᵢⱼ = Σₖ aᵢₖ × bₖⱼ = (baris i dari A) · (kolom j dari B) = a₁ᵢ×b₁ⱼ + a₂ᵢ×b₂ⱼ + ... + aₙᵢ×bₙⱼ
📖 Materi: Invers Matriks 2×2

Rumus Invers 2×2 — WAJIB HAFAL

Jika A = [[a, b], [c, d]], maka: A⁻¹ = 1/det(A) × [[d, -b], [-c, a]] di mana det(A) = ad - bc ≠ 0
⚠️ Ingat: Invers hanya ada jika det(A) ≠ 0. Jika det = 0, matriks disebut singular dan tidak memiliki invers.
📖 Materi: Transpose Matriks

Definisi Transpose

(Aᵀ)ᵢⱼ = Aⱼᵢ Artinya: baris menjadi kolom, kolom menjadi baris. Ordo: A(m×n) → Aᵀ(n×m)
Normal Soal 1 — Perkalian Matriks P×Q
P = [[2,1,-1],[3,0,4]] (2×3), Q = [[1,2],[0,-1],[3,1]] (3×2). Hitung P×Q.
  • 1
    Cek ordo: P(2×3) × Q(3×2) → Hasil berupa matriks 2×2 ✓ (kolom P = baris Q = 3)
  • 2
    Hitung C₁₁ = baris 1 P · kolom 1 Q
C₁₁= [2,1,-1]·[1,0,3] = 2·1 + 1·0 + (-1)·3 = 2+0-3 = -1
C₁₂= [2,1,-1]·[2,-1,1] = 2·2 + 1·(-1) + (-1)·1 = 4-1-1 = 2
C₂₁= [3,0,4]·[1,0,3] = 3·1 + 0·0 + 4·3 = 3+0+12 = 15
C₂₂= [3,0,4]·[2,-1,1] = 3·2 + 0·(-1) + 4·1 = 6+0+4 = 10
✅ P×Q = [[-1, 2], [15, 10]]
Easy Soal 2 — Invers Matriks 2×2
Cari invers dari M = [[5,2],[7,3]].
  • 1
    Hitung det(M):
det(M)= (5)(3) - (2)(7) = 15 - 14 = 1

    Terapkan rumus invers: Tukar a dan d, negatifkan b dan c, bagi dengan det
M = [[5,2],[7,3]] → swap a,d: [[3,2],[7,5]]
Negasi b,c: [[3, -2], [-7, 5]]
M⁻¹ = (1/1) × [[3,-2],[-7,5]] = [[3,-2],[-7,5]]
  • 3
    Verifikasi: M × M⁻¹ harus = I
[[5,2],[7,3]] × [[3,-2],[-7,5]]
(1,1): 5·3+2·(-7)=15-14=1   (1,2): 5·(-2)+2·5=-10+10=0
(2,1): 7·3+3·(-7)=21-21=0   (2,2): 7·(-2)+3·5=-14+15=1 ✓
✅ M⁻¹ = [[3, -2], [-7, 5]]

Determinan Matriks

Sarrus, determinan 2×2 dan 3×3, sifat matriks diagonal

📖 Materi: Determinan 2×2

Rumus Determinan 2×2

det([[a, b], [c, d]]) = ad - bc Diagonal utama (kiri-kanan) DIKURANG diagonal anti (kanan-kiri)
📖 Materi: Determinan 3×3 (Metode Sarrus)

Langkah Sarrus

  1. Tulis matriks 3×3, lalu tambahkan 2 kolom pertama di sisi kanan
  2. Jumlahkan 3 diagonal ↘ (kiri atas ke kanan bawah)
  3. Kurangi 3 diagonal ↙ (kanan atas ke kiri bawah)

Formula Sarrus

det = (a₁₁·a₂₂·a₃₃ + a₁₂·a₂₃·a₃₁ + a₁₃·a₂₁·a₃₂) − (a₁₃·a₂₂·a₃₁ + a₁₁·a₂₃·a₃₂ + a₁₂·a₂₁·a₃₃)
📖 Materi: Sifat Matriks Diagonal

Sifat Penting

Determinan matriks diagonal/segitiga = hasil kali elemen-elemen diagonal utamanya.

Contoh: D = [[5,0,0],[0,-3,0],[0,0,2]] → det(D) = 5×(−3)×2 = −30

Easy Soal 1 — Determinan 2×2
Hitung det(K) untuk K = [[6,-2],[4,1]].
Rumusdet = (a×d) − (b×c)
Diag ↘a×d = 6×1 = 6
Anti ↙b×c = (-2)×4 = -8
det= 6 − (-8) = 6 + 8 = 14
✅ det(K) = 14
Normal Soal 2 — Sarrus 3×3
Hitung det(L) dengan metode Sarrus. L = [[2,1,0],[1,-1,3],[0,2,4]].
  • 1
    Tulis matriks + 2 kolom ekstra di kanan:
L            | kol ekstra
2   1   0   |   2   1
1  -1   3   |   1  -1
0   2   4   |   0   2

    3 Diagonal ↘ (positif):
D1 ↘2×(-1)×4 = -8
D2 ↘1×3×0 = 0
D3 ↘0×1×2 = 0
Total (+)-8 + 0 + 0 = -8
  • 3
    3 Diagonal ↙ (negatif):
D1 ↙0×(-1)×0 = 0
D2 ↙2×3×2 = 12
D3 ↙4×1×1 = 4
Total (−)0 + 12 + 4 = 16
det(L)= (+) − (−) = -8 − 16 = -24

Minggu 5
Ekspansi Kofaktor & Aturan Cramer

Minor, kofaktor, ekspansi baris/kolom, Aturan Cramer

📖 Materi: Minor & Kofaktor

Definisi Minor Mᵢⱼ

Mᵢⱼ = determinan matriks yang diperoleh dengan menghapus baris i dan kolom j dari matriks A

Definisi Kofaktor Cᵢⱼ

Cᵢⱼ = (-1)^(i+j) × Mᵢⱼ Tanda kofaktor mengikuti pola checkerboard: [+ − + − ...] [− + − + ...] [+ − + − ...]
⚠️ Cara cepat cek tanda: Jika (i+j) genap → tanda +, jika (i+j) ganjil → tanda −
📖 Materi: Ekspansi Kofaktor

Ekspansi Sepanjang Baris i atau Kolom j

det(A) = Σⱼ aᵢⱼ × Cᵢⱼ (ekspansi baris i)
det(A) = Σᵢ aᵢⱼ × Cᵢⱼ (ekspansi kolom j)

Tips: Pilih baris/kolom yang paling banyak nolnya → perhitungan lebih singkat!

📖 Materi: Aturan Cramer

Aturan Cramer untuk Ax = b

xᵢ = det(Aᵢ) / det(A) di mana Aᵢ = matriks A dengan kolom ke-i diganti vektor b Syarat: det(A) ≠ 0
Normal Soal 1 — Minor & Kofaktor M₂₁, C₂₁
A = [[3,1,-4],[2,5,6],[1,4,8]]. Hitung minor M₂₁ dan kofaktor C₂₁.
  • 1
    Hitung M₂₁: Hapus baris 2 dan kolom 1 dari A
A = [[3,1,-4],[2,5,6],[1,4,8]]

Hapus baris 2, kolom 1 → sisa: [[1,-4],[4,8]]

M₂₁ = det([[1,-4],[4,8]]) = 1·8 − (-4)·4 = 8 + 16 = 24
  • 2
    Hitung C₂₁:
Tanda(-1)^(2+1) = (-1)^3 = -1
C₂₁= (-1) × M₂₁ = (-1) × 24 = -24
✅ M₂₁ = 24, C₂₁ = −24
Normal Soal 2 — Ekspansi Kofaktor Kolom 2
B = [[1,0,2],[3,4,-1],[5,0,6]]. Hitung det(B) dengan ekspansi kofaktor kolom 2.
  • 1
    Ambil elemen kolom 2: b₁₂=0, b₂₂=4, b₃₂=0
    Hanya b₂₂=4 ≠ 0 → hitung C₂₂ saja!
  • 2
    Hitung C₂₂:
Tanda(-1)^(2+2) = (-1)^4 = +1
M₂₂Hapus baris 2, kol 2 → [[1,2],[5,6]]
det([[1,2],[5,6]]) = 1·6 − 2·5 = 6 − 10 = -4
C₂₂= (+1) × (-4) = -4
  • 3
    Ekspansi kolom 2:
det(B)= b₁₂·C₁₂ + b₂₂·C₂₂ + b₃₂·C₃₂
= 0·C₁₂ + 4·(-4) + 0·C₃₂
= 0 + (-16) + 0 = -16
✅ det(B) = −16
Normal Soal 3 — Aturan Cramer (cari y)
Gunakan Aturan Cramer untuk cari y: 3x + y = 5, 2x − y = 0
  • 1
    Tentukan matriks A dan vektor b:
A = [[3,1],[2,-1]], b = [5, 0]
det(A)= 3·(-1) − 1·2 = -3 - 2 = -5
  • 2
    Bentuk A_y: Ganti kolom 2 (kolom y) dengan vektor b
A_y = [[3,5],[2,0]]
det(A_y)= 3·0 − 5·2 = 0 - 10 = -10
  • 3
    Terapkan Aturan Cramer:
y= det(A_y) / det(A) = -10 / -5 = 2
✅ y = 2

x = 10/5 = 2
Minggu 6
SPL Homogen & Kebebasan Linear

Solusi trivial/non-trivial, kombinasi linear, dependensi vektor

📖 Materi: SPL Homogen

Definisi SPL Homogen

SPL disebut homogen jika semua ruas kanan = 0, yaitu bentuk Ax = 0.

Sifat Penting SPL Homogen

1. SELALU konsisten (punya solusi) → minimal x = 0 (solusi trivial) 2. Solusi trivial: x₁ = x₂ = ... = xₙ = 0 3. Solusi non-trivial: ada variabel ≠ 0 4. Jika n > m (var > persamaan) → PASTI ada solusi non-trivial ∞
📖 Materi: Kebebasan Linear (Linear Independence)

Bebas Linear vs Bergantung Linear

Bebas Linear: Satu-satunya cara agar c₁v₁ + c₂v₂ + ... = 0 adalah c₁=c₂=...=0 (semua nol)

Bergantung Linear: Ada solusi non-trivial dari c₁v₁ + c₂v₂ + ... = 0 (ada cᵢ ≠ 0)

Cara cepat: Jika salah satu vektor adalah kelipatan skalar dari vektor lain → bergantung linear!
Jika ada vektor nol dalam himpunan → otomatis bergantung linear.
📖 Materi: Kombinasi Linear

Definisi

Vektor w adalah kombinasi linear dari v₁, v₂, ..., vₙ jika:
w = c₁v₁ + c₂v₂ + ... + cₙvₙ (untuk suatu konstanta c₁...cₙ)

Easy Soal 1 — SPL Homogen & Solusi Trivial
Jelaskan mengapa SPL homogen tidak mungkin inkonsisten, dan apa itu solusi trivial.
  • 1
    Mengapa tidak bisa inkonsisten?
    SPL homogen berbentuk Ax = 0. Substitusikan x = 0 (semua variabel = 0):
    A·0 = 0 ✓ → Ini selalu memenuhi persamaan, apapun matriks A-nya.
    Karena x = 0 selalu merupakan solusi, SPL homogen tidak pernah inkonsisten (selalu punya minimal 1 solusi).
  • 2
    Solusi Trivial
    Solusi trivial adalah solusi di mana semua variabel bernilai nol:
    x₁ = 0, x₂ = 0, ..., xₙ = 0
    Ini adalah solusi "paling sederhana" dan selalu ada pada setiap SPL homogen.
✅ SPL homogen tidak inkonsisten karena x=0 selalu solusi. Solusi trivial: semua variabel = 0.
Normal Soal 2 — Bebas/Bergantung Linear
Apakah u = [1,2,3] dan v = [-2,-4,-6] bebas linear atau bergantung linear?
  • 1
    Cek apakah satu vektor adalah kelipatan vektor lain:
v = [-2,-4,-6] = -2 × [1,2,3] = -2 × u
  • 2
    Pembuktian matematis: Cari c₁, c₂ ≠ (0,0) sehingga c₁u + c₂v = 0
c₁[1,2,3] + c₂[-2,-4,-6] = [0,0,0]
Pilih c₁=2, c₂=1: 2[1,2,3] + 1[-2,-4,-6]
= [2,4,6] + [-2,-4,-6] = [0,0,0] ✓
Ada solusi non-trivial (c₁=2, c₂=1) ≠ (0,0)
✅ Bergantung Linear — v = -2u, ada solusi non-trivial untuk c₁u + c₂v = 0
Easy Soal 3 — Kombinasi Linear
Nyatakan w = [5,9] sebagai kombinasi linear dari e₁ = [1,0] dan e₂ = [0,1].
Misalkanw = c₁·e₁ + c₂·e₂
[5,9] = c₁·[1,0] + c₂·[0,1]
[5,9] = [c₁, 0] + [0, c₂] = [c₁, c₂]
∴ c₁ = 5, c₂ = 9
w = 5·e₁ + 9·e₂
✅ w = 5e₁ + 9e₂ = 5[1,0] + 9[0,1]
e₁ dan e₂ adalah vektor basis standar (standard basis vectors)

L1. Apakah v₁=[1,0,0], v₂=[0,1,0], v₃=[0,0,1] bebas linear atau bergantung?

✅ Bebas Linear. Ini adalah vektor basis standar R³. Tidak ada yang merupakan kelipatan atau kombinasi dari yang lain. Satu-satunya solusi c₁v₁+c₂v₂+c₃v₃=0 adalah c₁=c₂=c₃=0.

Dari x₁+x₂=0 dan x₁-x₂=0
Tambahkan: 2x₁=0 → x₁=0
Substitusi: x₂=0
Hanya punya solusi trivial x₁=x₂=0.

Review: Matriks, SPL, Vektor

Polinomial matriks, metode invers, pernyataan konseptual aljabar

📖 Materi: Polinomial Matriks

Menghitung f(A) untuk Polinomial f

Jika f(t) = t² - 5t + 2, maka: f(A) = A² - 5A + 2I Hitung A² (A × A), lalu 5A, lalu 2I, kemudian kurangkan. I = matriks identitas dengan ordo sama dengan A.
📖 Materi: Menyelesaikan SPL dengan Metode Invers

Langkah-Langkah

Dari Ax = b: x = A⁻¹ · b 1. Hitung det(A) 2. Cari A⁻¹ (invers matriks) 3. Kalikan A⁻¹ dengan b
Normal Soal 1 — Polinomial Matriks: A² − 5A + 2I
A = [[1,2],[3,4]]. Hitung A² − 5A + 2I.
  • 1
    Hitung A²:
A² = A×A = [[1,2],[3,4]] × [[1,2],[3,4]]
(1,1)1·1 + 2·3 = 1+6 = 7
(1,2)1·2 + 2·4 = 2+8 = 10
(2,1)3·1 + 4·3 = 3+12 = 15
(2,2)3·2 + 4·4 = 6+16 = 22
A² = [[7,10],[15,22]]
  • 2
    Hitung 5A dan 2I:
5A= [[5,10],[15,20]]
2I= 2×[[1,0],[0,1]] = [[2,0],[0,2]]
  • 3
    Kombinasikan A² − 5A + 2I:
(1,1)7 − 5 + 2 = 4
(1,2)10 − 10 + 0 = 0
(2,1)15 − 15 + 0 = 0
(2,2)22 − 20 + 2 = 4
✅ A² − 5A + 2I = [[4, 0], [0, 4]] = 4I
(Menariknya, hasilnya adalah kelipatan matriks identitas!)
Normal Soal 2 — SPL dengan Metode Invers
Selesaikan: 4x₁ + x₂ = 14, 3x₁ + 2x₂ = 13. Gunakan x = A⁻¹b.
  • 1
    Identifikasi A dan b:
A = [[4,1],[3,2]], b = [14, 13]
  • 2
    Hitung det(A) dan A⁻¹:
det(A)= 4·2 − 1·3 = 8 − 3 = 5
A⁻¹= (1/5)·[[2,-1],[-3,4]]
  • 3
    Hitung x = A⁻¹·b:
x = (1/5)·[[2,-1],[-3,4]] · [14,13]
x₁= (1/5)·(2·14 + (-1)·13) = (1/5)·(28-13) = (1/5)·15 = 3
x₂= (1/5)·((-3)·14 + 4·13) = (1/5)·(-42+52) = (1/5)·10 = 2
  • 4
    Verifikasi: Substitusi ke persamaan asal
Cek 14(3) + 1(2) = 12+2 = 14 ✓
Cek 23(3) + 2(2) = 9+4 = 13 ✓
✅ x₁ = 3, x₂ = 2
Easy Soal 3 — Pernyataan Konseptual (Benar/Salah)
Tentukan Benar/Salah dan berikan alasan:
a. "Jika himpunan vektor mengandung ≥1 vektor nol → pasti bergantung linear."
b. "SPL homogen dengan variabel lebih banyak dari persamaan → pasti ada solusi non-trivial."
  • a
    ✅ BENAR
    Alasan: Misalkan 0 adalah vektor nol dalam himpunan S = {v₁, v₂, ..., 0, ...}.
    Maka: 0·v₁ + 0·v₂ + ... + 1·0 + ... = 0
    Ini adalah kombinasi linear yang menghasilkan vektor nol dengan koefisien vektor nol = 1 ≠ 0.
    Karena ada solusi non-trivial → himpunan tersebut bergantung linear.
  • b
    ✅ BENAR
    Alasan: Jika SPL homogen Ax = 0 punya n variabel dan m persamaan, dengan n > m:
    Matriks A berukuran m×n. Setelah eliminasi Gauss, maksimum ada m pivot.
    Karena n > m → ada minimal n−m variabel bebas (free variables).
    Variabel bebas dapat diberi nilai selain 0 → menghasilkan solusi non-trivial tak hingga.
✅ a. BENAR (vektor nol membuat kombinasi non-trivial)  |  b. BENAR (lebih banyak variabel = variabel bebas)
🎯 Soal Latihan Terakhir — Review Mingguan
R1. [Review M1] Sebutkan 3 contoh yang BUKAN persamaan linear, masing-masing berbeda alasannya.

1. x² + y = 3 → pangkat > 1
2. xy + z = 1 → perkalian antar variabel

R2. [Review M4] Jika det(A) = 0, apa artinya? Bisakah A memiliki invers?

Jika det(A) = 0, matriks A disebut singular. Matriks singular tidak memiliki invers karena rumus A⁻¹ = (1/det)×adj(A) → pembagi = 0 (tak terdefinisi). SPL Ax=b dengan A singular bisa tak konsisten atau punya ∞ solusi.

R3. [Review M5] Apa perbedaan Minor dan Kofaktor? Kapan keduanya bernilai sama?

Minor Mᵢⱼ = determinan submatriks setelah hapus baris i dan kolom j.
Kofaktor Cᵢⱼ = (-1)^(i+j) × Mᵢⱼ.
Keduanya bernilai sama ketika (i+j) genap → (-1)^(i+j) = +1, sehingga Cᵢⱼ = Mᵢⱼ.

📋 Cheat Sheet Rumus Wajib Hafal

Invers 2×2

A⁻¹ = (1/(ad-bc)) × [[d,-b],[-c,a]]

Sarrus (garis besar)

det = Σ(diag ↘) − Σ(diag ↙)

Kofaktor

Cᵢⱼ = (-1)^(i+j) × Mᵢⱼ

Aturan Cramer

xᵢ = det(Aᵢ) / det(A)

Det Diagonal

det = d₁₁ × d₂₂ × ... × dₙₙ

Metode Invers SPL

Ax = b → x = A⁻¹b
